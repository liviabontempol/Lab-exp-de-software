# Roteiro no PowerShell (VSC)

## Preparação do ambiente

### Criação do ambiente virtual:
python -m venv .venv

### Ativação do ambiente virtual:
.venv\Scripts\Activate.ps1

### Instalação das dependências:
pip3 install requests (biblioteca requests)
pip3 install numpy
